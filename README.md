## 🎯 프로그램 소개

### 개요
**재생타이어 ROI 계산기 v3.0**은 실제 데이터를 기반으로 재생타이어 도입의 투자 수익률(ROI)을 정확하게 예측하고, AI 기반 마케팅 전략을 제공하는 종합 분석 도구입니다.
포트폴리오를 위한 PoC 프로젝트입니다. 
빠른 개발을 위해 별도의 디렉토리 설계 없이 main code에 코드를 모두 집약하였으며 데이터는 실제데이터로 바꿔 사용이 가능합니다.



### 주요 특징
- 🔗 **실제 데이터 연동**: 정부 API, 실시간 가격 정보 연동
- 🤖 **AI/ML 기능**: 머신러닝 기반 연비 예측 및 개인화 전략
- 📊 **고급 분석**: 몬테카를로 시뮬레이션, NPV/IRR 계산
- 🌍 **ESG 평가**: 환경 영향 정량화 및 탄소 배출 계산
- 🎯 **마케팅 AI**: 고객별 맞춤 전략 및 세일즈 스크립트 생성

### 대상 사용자
- 운송업체 경영진 및 관리자
- 차량 관리 담당자
- 재생타이어 판매업체
- 환경 관련 정책 담당자

**지원 기능:**

- 실시간 타이어 가격 검색

**사용법:**
1. 사이드바에서 "실시간 차량 조회" 선택
2. 차량등록번호 입력 (예: 12가1234)
3. "🔎 실시간 조회" 버튼 클릭

### 2. 파일 업로드 분석
실제 주행 데이터를 업로드하여 더 정확한 분석을 수행합니다.

- `fuel_consumption`: 연료 소비량 (L)

**선택 컬럼:**
- `route_type`: 도로 유형 (도심/고속도로)
- `driver_id`: 운전자 식별자
- `weather`: 날씨 조건

**입력 정보:**
- 제조사 및 차량 분류
- 배기량 (cc)
- 차량 중량 (kg)
- 연식

**분석 변수:**

3가지 시나리오(보수적/기본/낙관적)로 미래 성과를 예측합니다.

## 📈 분석 결과 해석

### KPI 대시보드 해석

#### Enhanced ROI (향상된 투자수익률)
- **의미**: 재생타이어 투자 대비 수익률
- **계산**: (총 절감액 / 초기 투자액) × 100
- **판단 기준**:
    - 150% 이상: A+ 등급 (강력 추천)
    - 100-149%: A 등급 (추천)
    - 50-99%: B 등급 (검토 권장)
    - 50% 미만: C 등급 (신중 검토)

#### NPV (순현재가치)
- **의미**: 현재 가치로 환산한 순수익
- **계산**: 미래 현금흐름의 현재가치 - 초기 투자액
- **판단 기준**:
    - NPV > 0: 투자 가치 있음
    - NPV = 0: 손익분기점
    - NPV < 0: 투자 손실

#### IRR (내부수익률)
- **의미**: NPV가 0이 되는 할인율
- **계산**: NPV = 0이 되는 이자율
- **판단 기준**:
    - IRR > 요구수익률: 투자 매력적
    - IRR = 요구수익률: 손익분기점
    - IRR < 요구수익률: 투자 부적절

### ESG 점수 해석
- **80점 이상**: 우수 등급 (ESG 경영 기여도 높음)
- **60-79점**: 양호 등급 (적정 수준의 환경 기여)
- **40-59점**: 보통 등급 (최소한의 환경 효과)
- **40점 미만**: 미흡 등급 (환경 효과 제한적)

### 리스크 분석 해석

#### 신뢰도 수준
- **90% 이상**: 매우 높은 신뢰도
- **80-89%**: 높은 신뢰도
- **70-79%**: 보통 신뢰도
- **70% 미만**: 낮은 신뢰도 (추가 데이터 필요)

#### 손실 확률
- **5% 이하**: 매우 안전한 투자
- **5-10%**: 안전한 투자
- **10-20%**: 보통 위험
- **20% 이상**: 높은 위험

## 📚 전문 용어 해설

### 재무 분석 용어

#### ROI (Return On Investment)
**정의**: 투자 수익률
**설명**: 투자한 금액 대비 얻은 수익의 비율을 백분율로 나타낸 지표입니다. 재생타이어 도입으로 얻는 비용 절감 효과를 투자 금액으로 나누어 계산합니다.

#### NPV (Net Present Value)
**정의**: 순현재가치
**설명**: 미래에 발생할 현금흐름을 현재 가치로 환산했을 때의 순수익입니다. 할인율을 적용하여 돈의 시간 가치를 반영합니다.

#### IRR (Internal Rate of Return)
**정의**: 내부수익률
**설명**: 투자로부터 얻는 실제 수익률로, NPV가 0이 되는 할인율을 의미합니다. 은행 이자율과 비교하여 투자 매력도를 판단할 수 있습니다.

#### PI (Profitability Index)
**정의**: 수익성 지수
**설명**: 투자 1원당 얼마의 현재가치를 창출하는지를 나타내는 지표입니다. 1보다 크면 투자 가치가 있다고 판단합니다.

#### 페이백 기간 (Payback Period)
**정의**: 투자금 회수 기간
**설명**: 초기 투자액을 연간 현금흐름으로 나누어 계산한 투자금 회수에 필요한 시간입니다.

### 환경 분석 용어

#### CO₂ 배출 계수
**정의**: 연료 1리터당 배출되는 이산화탄소량
**설명**:
- 경유: 2.68 kg CO₂/L
- 휘발유: 2.31 kg CO₂/L
- LPG: 1.87 kg CO₂/L

#### ESG (Environmental, Social, Governance)
**정의**: 환경, 사회, 지배구조
**설명**: 기업의 지속가능성을 평가하는 비재무적 요소입니다. 재생타이어 도입은 환경(E) 측면에서 긍정적 영향을 미칩니다.

#### 탄소배출권
**정의**: 온실가스 배출 권리를 거래할 수 있는 권리증서
**설명**: CO₂ 감축량을 금전적 가치로 환산할 때 사용하며, 국내 기준 톤당 약 15,000원입니다.

### 기술 분석 용어

#### 몬테카를로 시뮬레이션
**정의**: 확률적 시뮬레이션 기법
**설명**: 불확실한 변수들에 대해 무작위 값을 생성하여 수천 번의 시나리오를 실행해 결과의 확률 분포를 구하는 방법입니다.

#### 민감도 분석
**정의**: 변수 변화에 대한 결과 민감도 분석
**설명**: 주요 입력 변수가 ±20% 변할 때 최종 결과에 미치는 영향을 분석하여 가장 중요한 변수를 식별합니다.

### 재생타이어 전문 용어

#### 재생타이어 (Retread Tire)
**정의**: 중고 타이어에 새로운 트레드를 부착한 타이어
**설명**: 타이어 케이싱(골격)을 재사용하고 접지면만 새로 만든 타이어로, 신품 대비 50% 가격에 95% 성능을 제공합니다.

#### 케이싱 (Casing)
**정의**: 타이어의 골격 부분
**설명**: 타이어의 내부 구조로, 재생타이어 제작 시 재사용되는 부분입니다.

#### 최대 재생 횟수
**정의**: 한 케이싱으로 재생 가능한 최대 횟수
**설명**: 일반적으로 승용차 타이어는 1-2회, 트럭용 타이어는 2-3회 재생이 가능합니다.

## 🧮 계산 공식 및 알고리즘

### 1. 기본 ROI 계산

#### 직접 비용 절감
```
신품 타이어 총비용 = 타이어 교체 횟수 × 타이어 개수 × 신품 가격 + 작업비 + 처리비
재생 타이어 총비용 = (재생 횟수 × 재생 가격 + 잔여 신품 교체 × 신품 가격) × 타이어 개수 + 작업비 + 처리비
직접 절감액 = 신품 총비용 - 재생 총비용
```

#### 연비 개선 효과
```
기본 연료 소비량 = 총 주행거리 ÷ 기본 연비
개선 연료 소비량 = 총 주행거리 ÷ (기본 연비 × 1.025)  // 2.5% 개선
연료 절감액 = (기본 소비량 - 개선 소비량) × 유가
```

#### 종합 ROI
```
총 절감액 = 직접 절감액 + 연료 절감액 + 정비비 절감액
초기 투자액 = 타이어 개수 × 재생타이어 가격
ROI (%) = (총 절감액 ÷ 초기 투자액) × 100
```

### 2. NPV 계산

```
NPV = Σ[t=1 to n] (연간 현금흐름t ÷ (1 + 할인율)^t) - 초기 투자액

여기서:
- t: 연도 (1, 2, 3, 4, 5)
- n: 분석 기간 (5년)
- 할인율: 5% (정부 기준금리 + 위험 프리미엄)
```

### 3. IRR 계산

IRR은 다음 방정식을 만족하는 이자율입니다:
```
0 = Σ[t=0 to n] (현금흐름t ÷ (1 + IRR)^t)

수치해석 방법으로 근삿값 계산:
IRR ≈ (연간 현금흐름 ÷ 초기 투자액) × 100  // 단순화된 추정
```

### 4. ML 연비 예측 모델

선형 회귀 모델을 사용한 연비 예측:
```
예측 연비 = 절편 + (배기량 × 계수1) + (중량 × 계수2) + (연식 × 계수3) + 차량타입 가중치

모델 계수:
- 절편: 12.5
- 배기량 계수: -0.002
- 중량 계수: -0.0015
- 연식 계수: 0.1 (2015년 기준)
- 차량타입 가중치:
  * 승용: +3.0
  * 화물: -1.5
  * 승합: -2.8
```

### 5. 환경 영향 계산

#### CO₂ 감축량
```
절약 연료량 = 총 주행거리 × 연비 개선율 ÷ 기본 연비
CO₂ 감축량 = 절약 연료량 × 배출 계수

배출 계수:
- 경유: 2.68 kg CO₂/L
- 휘발유: 2.31 kg CO₂/L
- LPG: 1.87 kg CO₂/L
```

#### 폐기물 감소
```
절약 타이어 수 = min(타이어 교체 횟수, 최대 재생 횟수) × 타이어 개수
폐기물 감소량 = 절약 타이어 수 × 8.5 kg/개
```

#### ESG 점수 계산
```
ESG 점수 = min(100, 
    (CO₂ 감축량 ÷ 100) × 20 +     // CO₂ 영향 (20점)
    (폐기물 감소량 ÷ 50) × 25 +    // 폐기물 영향 (25점)
    (절약 타이어 수 ÷ 10) × 15 +   // 자원 절약 (15점)
    40                             // 기본 점수 (40점)
)
```

### 6. 위험도 분석

#### 총 위험 점수
```
총 위험 점수 = 
    유가 변동성 × 0.3 +           // 30% 가중치
    타이어 가격 변동성 × 0.2 +     // 20% 가중치
    재생타이어 품질 위험 × 0.15 +  // 15% 가중치
    공급망 위험 × 0.15 +          // 15% 가중치
    도입 저항 위험 × 0.2          // 20% 가중치
```

#### 위험 조정 ROI
```
위험 조정 ROI = 기본 ROI × (1 - 총 위험 점수)
```

### 7. 몬테카를로 시뮬레이션

1,000회 시뮬레이션을 통한 ROI 분포 계산:
```python
for i in range(1000):
    유가_변동 = random.normal(1.0, 0.15)    # 평균 1.0, 표준편차 0.15
    타이어가격_변동 = random.normal(1.0, 0.10)  # 평균 1.0, 표준편차 0.10
    주행거리_변동 = random.normal(1.0, 0.20)   # 평균 1.0, 표준편차 0.20
    
    조정_ROI = 기본_ROI × (
        0.4 × 유가_변동 +      # 40% 영향
        0.3 × 타이어가격_변동 + # 30% 영향
        0.3 × 주행거리_변동    # 30% 영향
    )
    
    ROI_분포.append(max(0, 조정_ROI))  # 음수 방지
```

### 8. 민감도 분석

각 변수의 ±20% 변동이 ROI에 미치는 영향:
```
for 변수 in [주행거리, 유가, 타이어가격, 연비개선율, 할인율]:
    양의_영향 = 기본_ROI × (1 + 0.2 × 영향_계수)
    음의_영향 = 기본_ROI × (1 - 0.2 × 영향_계수)
    민감도_점수 = abs(양의_영향 - 음의_영향)
```
